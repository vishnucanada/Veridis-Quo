# Veridis-Quo
Deepmind Released a dataset that contains mathematics problem ranging from arithmetic, algebra, calculus and more. The reasoning is that LLM's are not answering with the reasoning otherwise it would be able to solve easy problems but the higher in complexity of the problem the more discrepency can be seen.

Veridis-Quo is a expiremental project to determine how to create a reasoning model for an LLM, we will compare with DeepSeek R1 which claims reasoning capabilities with its RL enforcement.
